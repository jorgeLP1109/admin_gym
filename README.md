# LEVEL UP S&A CENTER - Sistema Administrativo

Sistema de gestión integral para gimnasio con control de estudiantes, clases, pagos y contabilidad.

## 🚀 Tecnologías

- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Node.js + Express + PostgreSQL
- **Autenticación**: JWT
- **Hosting**: Vercel (Frontend) + Supabase (Backend + DB)

## 📦 Instalación

```bash
# Instalar todas las dependencias
npm run install:all

# Desarrollo
npm run dev
```

## 🎯 Módulos

- ✅ Autenticación y usuarios
- ✅ Gestión de estudiantes (con ficha médica)
- ✅ Gestión de profesores
- ✅ Gestión de clases (horarios, precios)
- ✅ Inscripciones
- ✅ Control de pagos (quincenal/mensual)
- ✅ Contabilidad (ingresos/gastos)
- ✅ Reportes y consultas
